## 摘要
我们提出了 TC-AE，一种基于 ViT 的深度压缩自编码器架构。现有方法通常通过增加潜表示（latent representations）的通道数，以在高压缩比下维持重建质量。然而，这种策略往往会导致潜表示崩溃，从而降低生成性能。TC-AE 并非依赖日益复杂的架构或多阶段训练方案，而是从 Token 空间（像素与图像潜变量之间的关键桥梁）的角度出发，通过两项互补的创新来应对这一挑战：首先，我们在固定潜变量预算下，通过调整 ViT 中的 patch 大小来研究 Token 数量的缩放，并发现激进的 Token 到潜变量压缩是限制有效缩放的关键因素。为解决这一问题，我们将 Token 到潜变量的压缩分解为两个阶段，从而减少了结构信息的损失，并实现了面向生成的有效 Token 数量缩放。其次，为了进一步缓解潜表示崩溃，我们通过联合自监督训练增强了图像 Token 的语义结构，从而产生了更利于生成的潜变量。凭借这些设计，TC-AE 在深度压缩下实现了显著提升的重建和生成性能。我们希望本研究能推动用于视觉生成的基于 ViT 的分词器（tokenizer）的发展。

## Abstract
We propose TC-AE, a ViT-based architecture for deep compression autoencoders. Existing methods commonly increase the channel number of latent representations to maintain reconstruction quality under high compression ratios. However, this strategy often leads to latent representation collapse, which degrades generative performance. Instead of relying on increasingly complex architectures or multi-stage training schemes, TC-AE addresses this challenge from the perspective of the token space, the key bridge between pixels and image latents, through two complementary innovations: Firstly, we study token number scaling by adjusting the patch size in ViT under a fixed latent budget, and identify aggressive token-to-latent compression as the key factor that limits effective scaling. To address this issue, we decompose token-to-latent compression into two stages, reducing structural information loss and enabling effective token number scaling for generation. Secondly, to further mitigate latent representation collapse, we enhance the semantic structure of image tokens via joint self-supervised training, leading to more generative-friendly latents. With these designs, TC-AE achieves substantially improved reconstruction and generative performance under deep compression. We hope our research will advance ViT-based tokenizer for visual generation.

*   **核心问题**：在潜扩散模型（LDMs）中，为了提高效率，研究者倾向于使用“深度压缩”分词器（Tokenizer），即极度压缩潜表示（Latent Representation）的空间分辨率，并通过增加通道数来补偿信息损失。然而，这种**极端的空间压缩与通道扩张会导致“潜表示崩溃”**（Latent Representation Collapse），严重损害生成图像的质量。
*   **研究动机**：现有的基于 ViT 的分词器主要关注模型参数量的缩放，而忽视了连接像素与潜空间的桥梁——**Token 空间**。作者发现，简单地增加 Token 数量（减小 Patch Size）虽能提升重建质量，但由于 Token 到潜变量之间的压缩过于激进，会导致严重的结构信息丢失，反而使生成性能下降。

### 2. 方法论
TC-AE 提出了两项核心创新来优化 Token 空间：
*   **分阶段 Token 压缩（Staged Token Compression, STC）**：
    *   **核心思想**：将原本在 Bottleneck 处的一次性激进压缩分解为多个阶段。
    *   **技术细节**：ViT 编码器被分为两个阶段。第一阶段在较小的 Patch Size（如 $p=8$）下处理大量 Token 以捕获细节；随后通过一个中间压缩层（如卷积模块）将 Token 序列长度缩减（如 1/4）；第二阶段处理压缩后的 Token，最后再通过最终 Bottleneck 输出潜变量。这种设计缓解了结构信息的突发性丢失。
*   **联合自监督训练（Joint Self-Supervised Training）**：
    *   **核心思想**：在训练分词器的同时，引入自监督学习（SSL）目标，强制 Token 学习语义结构。
    *   **技术细节**：集成 **iBOT** 目标函数，通过掩码图像建模（MIM）和自蒸馏（Student-Teacher 架构）来增强 Token 的语义一致性。
*   **总损失函数**：$L_{TC-AE} = \alpha L_{rec} + L_{iBOT}$。其中 $L_{rec}$ 包含像素级 $L_1$ 损失、感知损失和对抗损失。

### 6. 主要结论与发现
*   **Token 缩放优于参数缩放**：在相同的计算预算下，增加 Token 数量（减小 Patch Size）比增加模型参数量对生成性能的提升更显著。
*   **STC 是关键**：分阶段压缩能有效保留潜空间的语义结构，解决深度压缩下的表示崩溃问题。

### 7. 优点
*   **视角新颖**：首次从 Token 空间容量的角度解决深度压缩自编码器的瓶颈问题。
*   **高效性**：在极低的计算开销下实现了卓越的生成效果，非常适合高分辨率图像生成的预处理。
*   **即插即用**：提出的 STC 和联合 SSL 训练方案具有普适性，可推广至其他基于 ViT 的架构。

### 8. 不足与局限
*   **重建与生成的权衡**：引入 SSL 虽然大幅提升了生成质量，但在某些指标上（如 PSNR）会导致重建精度轻微下降（尽管视觉上差异不大）。

*   **超参数敏感性**：$\alpha$（重建与 SSL 的权重比）对结果影响较大，需要针对不同任务进行精细调优。
