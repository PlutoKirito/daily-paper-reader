## 摘要
高效的图像压缩依赖于对局部和全局冗余的建模。大多数最先进的（SOTA）学习型图像压缩（LIC）方法基于卷积神经网络（CNN）或 Transformer，这些方法本质上是僵化的。标准的 CNN 卷积核和基于窗口的注意力机制施加了固定的感受野和静态的连接模式，这可能仅仅因为欧几里得空间中的邻近性而将非冗余像素耦合在一起。这种僵化限制了模型自适应捕获图像中随空间变化的冗余的能力，特别是在全局层面。为了克服这些局限性，我们提出了一种基于图神经网络（GNN）的内容自适应图像压缩框架。具体而言，我们的方法构建了双尺度图，从而实现了灵活的、数据驱动的感受野。此外，我们通过根据局部内容复杂度动态调整每个节点的邻居数量，引入了自适应连接性。这些创新使我们的基于图的学习型图像压缩（GLIC）模型能够有效地建模图像中多样的冗余模式，从而实现更高效、更具自适应性的压缩。实验表明，GLIC 达到了最先进的性能，在 Kodak、Tecnick 和 CLIC 数据集上相对于 VTM-9.1 分别实现了 19.29%、21.69% 和 18.71% 的 BD-rate 降低。代码将在 https://github.com/UnoC-727/GLIC 发布。

## Abstract
Efficient image compression relies on modeling both local and global redundancy. Most state-of-the-art (SOTA) learned image compression (LIC) methods are based on CNNs or Transformers, which are inherently rigid. Standard CNN kernels and window-based attention mechanisms impose fixed receptive fields and static connectivity patterns, which potentially couple non-redundant pixels simply due to their proximity in Euclidean space. This rigidity limits the model's ability to adaptively capture spatially varying redundancy across the image, particularly at the global level. To overcome these limitations, we propose a content-adaptive image compression framework based on Graph Neural Networks (GNNs). Specifically, our approach constructs dual-scale graphs that enable flexible, data-driven receptive fields. Furthermore, we introduce adaptive connectivity by dynamically adjusting the number of neighbors for each node based on local content complexity. These innovations empower our Graph-based Learned Image Compression (GLIC) model to effectively model diverse redundancy patterns across images, leading to more efficient and adaptive compression. Experiments demonstrate that GLIC achieves state-of-the-art performance, achieving BD-rate reductions of 19.29%, 21.69%, and 18.71% relative to VTM-9.1 on Kodak, Tecnick, and CLIC, respectively. Code will be released at https://github.com/UnoC-727/GLIC.

### 1. 核心问题与研究动机
*   **核心问题**：传统的学习型图像压缩（LIC）模型主要基于 CNN 或 Transformer 架构，这些架构具有**“僵理性”（Rigidity）**。
    *   **固定感受野**：CNN 卷积核和窗口注意力机制采用固定的空间连接模式，仅根据欧几里得距离耦合像素。
    *   **静态连接**：无法根据图像内容的复杂程度（如平滑区与纹理区）动态调整连接强度和范围，导致难以有效捕捉空间变化的冗余，尤其是在全局尺度上。
*   **研究背景**：为了突破固定窗口的限制，研究者开始探索图神经网络（GNN），因为 GNN 具有动态、数据驱动的连接特性，能捕捉非欧几里得关系。

### 2. 核心方法论
论文提出了 **GLIC（Graph-based Learned Image Compression）** 框架，其核心是 **图特征聚合（GFA）块**，包含以下关键技术：
*   **双尺度图构建（Dual-scale Graphs）**：
    *   **密集局部采样**：在固定窗口内采集候选节点，捕捉精细的局部结构。
    *   **稀疏全局采样**：通过网格步长（Mesh-grid）在全图范围内采样，以近线性的复杂度实现全局感受野。
*   **内容自适应邻居配额分配**：
    *   **RMS-Gradient 评分**：利用 Sobel 算子计算梯度，并通过均方根（RMS）跨通道池化得到复杂度分数。
    *   **动态度数（Degree）**：结构复杂的区域（高分）分配更多的邻居配额，平滑区域（低分）分配较少，从而实现像素级的计算资源自适应分配。
*   **基于二分查找的图构建**：
    *   计算节点间的余弦相似度，利用**二分查找算法**快速为每个节点寻找最优的相似度阈值，确保连接数符合分配的配额。
*   **边缘条件聚合**：
    *   采用边缘加权的特征更新机制，根据像素间的相似度动态聚合信息。

### 6. 主要结论与发现
*   **性能领先**：GLIC 在所有测试集上均显著优于 VTM-9.1，BD-rate 分别降低了 **19.29% (Kodak)**、**21.69% (Tecnick)** 和 **18.71% (CLIC)**。

*   **自适应能力**：可视化证明 GLIC 能根据图像语义（如背景 vs. 前景）自动调整感受野形状，实现了真正的“内容自适应”冗余消除。

### 7. 优点与亮点
*   **打破僵理性**：成功将 GNN 引入 LIC，解决了 CNN/Transformer 固定连接的问题。
*   **线性复杂度**：通过双尺度采样和二分查找，将原本 $O(N^2)$ 的全局图计算降低到近线性复杂度，使其具备处理高分辨率图像的实用性。
*   **精细化建模**：提出的 RMS-G 评分机制比传统的残差代理更能准确反映图像的编码难度。

### 8. 不足与局限
*   **实现复杂度**：尽管计算复杂度是线性的，但图构建过程（相似度计算、二分查找）在底层硬件（如专用芯片）上的实现可能比标准卷积更复杂。
*   **超参数敏感性**：平均度数（Average Degree）和采样窗口大小等超参数可能需要针对不同类型的图像进行微调以达到最优平衡。
*   **端到端优化限制**：虽然图构建是动态的，但图的拓扑结构在反向传播时通常是不可微的（本文通过相似度权重缓解了此问题），这可能限制了模型学习更极端连接模式的能力。
