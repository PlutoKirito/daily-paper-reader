## 摘要
我们提出了 TC-AE，一种基于 ViT 的深度压缩自编码器架构。现有方法通常通过增加潜表示（latent representations）的通道数，以在高压缩比下维持重建质量。然而，这种策略往往会导致潜表示崩溃，从而降低生成性能。TC-AE 并非依赖日益复杂的架构或多阶段训练方案，而是从 Token 空间（连接像素与图像潜变量的关键桥梁）的角度出发，通过两项互补的创新来应对这一挑战：首先，我们在固定潜变量预算下，通过调整 ViT 中的 patch 大小来研究 Token 数量的缩放，并确定激进的 Token 到潜变量压缩是限制有效缩放的关键因素。为解决此问题，我们将 Token 到潜变量的压缩分解为两个阶段，减少了结构信息损失，并实现了面向生成的有效 Token 数量缩放。其次，为了进一步缓解潜表示崩溃，我们通过联合自监督训练增强了图像 Token 的语义结构，从而产生更利于生成的潜变量。凭借这些设计，TC-AE 在深度压缩下显著提升了重建和生成性能。我们希望本研究能推动用于视觉生成的基于 ViT 的分词器（tokenizer）的发展。

## Abstract
We propose TC-AE, a ViT-based architecture for deep compression autoencoders. Existing methods commonly increase the channel number of latent representations to maintain reconstruction quality under high compression ratios. However, this strategy often leads to latent representation collapse, which degrades generative performance. Instead of relying on increasingly complex architectures or multi-stage training schemes, TC-AE addresses this challenge from the perspective of the token space, the key bridge between pixels and image latents, through two complementary innovations: Firstly, we study token number scaling by adjusting the patch size in ViT under a fixed latent budget, and identify aggressive token-to-latent compression as the key factor that limits effective scaling. To address this issue, we decompose token-to-latent compression into two stages, reducing structural information loss and enabling effective token number scaling for generation. Secondly, to further mitigate latent representation collapse, we enhance the semantic structure of image tokens via joint self-supervised training, leading to more generative-friendly latents. With these designs, TC-AE achieves substantially improved reconstruction and generative performance under deep compression. We hope our research will advance ViT-based tokenizer for visual generation.

#### 1. 核心问题与整体含义
在潜扩散模型（LDMs）中，为了提高效率，研究者倾向于使用“深度压缩”分词器（Tokenizer），即大幅降低潜表示（Latent）的空间分辨率。传统做法是通过增加通道数来补偿空间信息的损失，但这往往导致“潜表示崩溃”（Latent Representation Collapse），即重建质量尚可但生成性能大幅下降。
本文从 **Token 空间**（像素与潜变量之间的桥梁）的角度切入，指出基于 ViT 的分词器在深度压缩时，Token 到潜变量的单一激进压缩是导致性能瓶颈的主因。

#### 2. 论文提出的方法论
TC-AE（Token Capacity Autoencoder）通过两项核心创新来优化 Token 空间：
*   **分阶段 Token 压缩（Staged Token Compression, STC）：**
    *   **核心思想：** 避免在 Encoder 末端进行一次性的剧烈压缩。
    *   **技术细节：** 将 ViT 编码器分为两个阶段。第一阶段在较小的 patch size（如 $p=8$）下处理大量 Token 以捕获细节；随后通过一个中间瓶颈层（如卷积模块）将 Token 数量减少（如减为 1/4）；第二阶段继续处理压缩后的 Token，最后通过末端瓶颈层生成最终潜变量。这种“缓冲”机制减少了结构信息的突发性丢失。
*   **联合自监督训练（Joint Self-Supervised Learning）：**
    *   **核心思想：** 在训练分词器的同时，引入自监督任务来增强 Token 的语义结构。
    *   **技术细节：** 集成了 **iBOT** 目标函数，通过学生-教师网络的蒸馏机制（包括掩码图像建模 MIM 和类 Token 蒸馏），使编码器生成的 Token 具有更强的语义规律性，从而产生更利于生成模型学习的潜空间。

#### 6. 主要结论与发现
*   **Token 缩放悖论：** 在固定潜变量预算下，简单增加 Token 数量（减小 patch size）能提升重建，但会因瓶颈压缩过猛而损害生成性能。
*   **STC 的有效性：** 分阶段压缩能有效缓解上述悖论，使生成性能随 Token 数量增加而提升。
*   **语义结构的重要性：** 引入自监督学习能显著加速扩散模型的收敛速度（最高达 4.7 倍），并提升最终生成质量。

#### 7. 优点
*   **视角新颖：** 首次从 Token 空间容量而非单纯的通道或参数维度解决深度压缩问题。
*   **高效性：** 在极低的空间分辨率（仅 64 个 Token）下达到了与高分辨率分词器相当甚至更好的生成效果，且计算开销更低。
*   **无需外部预训练：** 通过联合训练实现语义增强，避免了对外部大规模预训练模型（如 DINOv2 权重）的强依赖。

#### 8. 不足与局限
*   **重建与生成的权衡：** 引入自监督学习（SSL）虽然大幅提升了生成性能，但在某些指标下会轻微降低重建的像素级保真度（尽管在小 patch size 下此影响减弱）。
*   **任务覆盖：** 实验主要集中在 ImageNet 类条件生成上，尚未在更大规模的文本到图像（T2I）数据集或视频生成任务中验证泛化性。
*   **超参数敏感性：** 分阶段压缩的切分位置（M 层）和自监督损失权重（$\alpha$）需要针对不同架构进行调优。
