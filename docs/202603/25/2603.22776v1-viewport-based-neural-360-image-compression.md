## 摘要
鉴于 360° 图像在社交媒体平台上的普及，360° 图像压缩已成为媒体存储和传输的关键技术。传统的 360° 图像压缩流程将球面图像投影到单个 2D 平面上，从而导致过采样和失真问题。在本文中，我们提出了一种新型的基于视口的 360° 图像神经压缩流程。通过将传统 360° 图像压缩流程中的图像投影替换为视口提取并高效压缩多个视口，所提流程最大限度地减少了固有的过采样和失真问题。然而，视口提取阻碍了压缩过程中多个视口之间的信息共享，导致球面图像全局信息的丢失。为了解决这一全局信息丢失问题，我们设计了一个神经视口编解码器，用于捕获跨多个视口的全局先验信息并最大限度地压缩视口数据。该视口编解码器由基于 Transformer 的视口上下文（VPCT）模块赋能，该模块可以与规范的基于学习的 2D 图像压缩结构集成。我们将所提流程与现有的 360° 图像压缩模型，以及基于学习的 2D 图像编解码器和标准手工编解码器构建的传统 360° 图像压缩流程进行了比较。结果表明，在不损失质量的情况下，与性能最好的 360° 图像压缩方法相比，我们的流程平均节省了 14.01% 的比特消耗。所提出的基于 VPCT 的编解码器在基于视口的神经压缩流程中也优于现有的 2D 图像编解码器。我们的代码可在以下网址找到：https://github.com/Jingwei-Liao/VPCT。

## Abstract
Given the popularity of 360° images on social media platforms, 360° image compression becomes a critical technology for media storage and transmission. Conventional 360° image compression pipeline projects the spherical image into a single 2D plane, leading to issues of oversampling and distortion. In this paper, we propose a novel viewport-based neural compression pipeline for 360° images. By replacing the image projection in conventional 360° image compression pipelines with viewport extraction and efficiently compressing multiple viewports, the proposed pipeline minimizes the inherent oversampling and distortion issues. However, viewport extraction impedes information sharing between multiple viewports during compression, causing the loss of global information about the spherical image. To tackle this global information loss, we design a neural viewport codec to capture global prior information across multiple viewports and maximally compress the viewport data. The viewport codec is empowered by a transformer-based ViewPort ConText (VPCT) module that can be integrated with canonical learning-based 2D image compression structures. We compare the proposed pipeline with existing 360° image compression models and conventional 360° image compression pipelines building on learning-based 2D image codecs and standard hand-crafted codecs. Results show that our pipeline saves an average of $14.01\%$ bit consumption compared to the best-performing 360° image compression methods without compromising quality. The proposed VPCT-based codec also outperforms existing 2D image codecs in the viewport-based neural compression pipeline. Our code can be found at: https://github.com/Jingwei-Liao/VPCT.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：传统的 360° 图像压缩通常采用“投影+2D压缩”的管线（如等距柱状投影 ERP）。这种方式存在两个固有缺陷：
    1.  **过采样（Oversampling）**：高纬度地区被过度拉伸，引入大量冗余像素。
    2.  **几何失真（Distortion）**：球面内容投影到平面后形状扭曲（如直线变曲线），降低了 2D 编解码器的重建质量。
*   **研究背景**：随着全景相机和 VR 设备的普及，360° 图像的存储和传输压力巨大。现有的神经压缩模型虽有改进，但大多仍基于 ERP 投影，无法从根本上消除上述投影带来的负面影响。

### 2. 论文提出的方法论
*   **核心思想**：提出一种**基于视口（Viewport-based）**的神经压缩管线。通过提取多个离散的、低失真、无过采样的视口来覆盖整个球面，从而替代传统的全局投影。
*   **关键技术细节**：
    *   **视口提取（Viewport Extraction）**：将球面局部区域近似为 2D 平面，最小化几何失真。
    *   **神经视口编解码器**：针对视口间存在重叠且缺乏全局关联的问题，设计了 **VPCT（ViewPort ConText）** 模块。
    *   **VPCT 模块结构**：
        *   **Intra-view Block（视图内块）**：利用带掩码（Masked）的自注意力机制捕获单个视口内的因果上下文信息。
        *   **Inter-view Block（视图间块）**：利用交叉视图注意力（Cross-view Attention）捕获当前视口与已编码视口之间的相关性，消除重叠区域的冗余。
    *   **集成方式**：VPCT 作为一个插件，可以无缝集成到现有的 2D 神经压缩结构（如 Factorized, Hyperprior, Joint, Reference）的熵模型中，通过特征融合层（Feature Fusion Layer）提升概率估计精度。

*   **性能领先**：在相同质量下，该方法比目前最优的 360° 图像压缩模型平均节省了 **14.01%** 的比特率。
*   **兼容性强**：VPCT 模块能显著提升所有测试的 2D 神经编解码器在全景场景下的表现。

### 7. 优点
*   **思路创新**：打破了“必须先投影成一张大图再压缩”的思维定式，从预处理阶段解决失真问题。
*   **模块化设计**：VPCT 模块不改变原 2D 编解码器的主干结构，具有极强的通用性和可扩展性。
*   **效率平衡**：在提升压缩率的同时，计算复杂度的增加非常微小，且支持并行加速。

### 8. 不足与局限
*   **顺序依赖**：由于采用自回归式的视图间预测，第一个编码的视口无法享受全局信息增益，导致不同视口的比特分配不均。
*   **FoV 敏感性**：视口的大小（FoV）对性能有显著影响，过大则失真严重，过小则重叠过多，目前仍需手动设定最优 FoV。
*   **极端场景**：在极高分辨率或极多视口的情况下，Transformer 的注意力机制可能会带来额外的显存压力（尽管文中显示目前 8K 场景表现良好）。
