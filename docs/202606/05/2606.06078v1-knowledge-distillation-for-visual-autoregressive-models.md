## 摘要
自回归图像生成模型表达能力强但计算量大，因此需要有效的模型压缩。知识蒸馏是一种自然的模型压缩方法，并在语言建模中得到了广泛研究，但其在视觉自回归生成中的行为尚未得到充分探索。在这项工作中，我们首次对自回归图像模型的蒸馏策略进行了系统研究。我们的分析表明，虽然标准蒸馏可以带来有意义的提升，但近期为语言模型开发的方法无法直接迁移到图像领域：长解码步长和视觉标记的模糊性使得教师监督不可靠，尤其是在学生条件上下文下。为了解决这一问题，我们提出了VarKD，一种针对视觉自回归模型的蒸馏框架，它在学生样本上进行蒸馏，同时选择性地应用教师监督并减少标记级别的模糊性。在ImageNet上基于多个自回归骨干网络的实验表明，VarKD持续优于先前的蒸馏基线，缩小了与大规模模型的差距。

## Abstract
Autoregressive (AR) image generation models are highly expressive but computationally intensive, motivating effective model compression. Knowledge distillation (KD) is a natural approach for model compression and has been widely studied in language modeling, yet its behavior in visual AR generation remains underexplored. In this work, we present the first systematic study of distillation strategies for AR image models. Our analysis shows that while standard distillation can yield meaningful gains, recent methods developed for language do not directly transfer to images: long decoding horizons and visual token ambiguity make teacher supervision unreliable especially under student-conditioned contexts. To address this, we propose VarKD, a distillation framework for visual autoregressive models that distills on student samples while selectively applying teacher supervision and reducing token-level ambiguity. Experiments on ImageNet across multiple AR backbones show that VarKD consistently outperforms prior distillation baselines, narrowing the gap to large-scale models.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：自回归（AR）图像生成模型（如 LlamaGen、ARPG）表达能力强，但计算成本极高，因为图像合成需要长序列的逐 token 解码（通常数百到上千步），导致推理延迟高，且小型模型因训练-推理不匹配（exposure bias）而性能严重退化。  
- **背景**：知识蒸馏（KD）在语言建模中已被广泛用于模型压缩，但直接迁移到视觉 AR 模型效果不佳。视觉 AR 模型有两个独有挑战：
  - **长解码序列**：图像 token 序列远长于文本，误差累积更严重。
  - **视觉 token 歧义**：离散码本中有大量视觉相似的 token，使教师模型的预测分布模糊，尤其在学生自生成上下文上监督信号不可靠。  
- **目标**：首次系统研究视觉 AR 模型的蒸馏策略，并针对上述挑战提出专用框架 VarKD，以有效压缩模型并缩小与大规模教师的差距。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：在**学生自生成样本**上进行蒸馏（on-policy），但通过**混合数据-学生上下文**、**置信度重加权**和**压缩空间蒸馏**来提升教师监督的可靠性。
- **关键技术细节**：
  1. **混合数据-学生上下文**：并非完全自由运行学生，而是用数据前缀（前 k 个真实 token）锚定，之后由学生自回归生成后缀。这平衡了训练-推理一致性。
  2. **选择性教师监督（熵重加权）**：根据教师预测分布的归一化熵计算置信度 α，低熵（高置信）的 token 获得高权重，高熵（模糊）的 token 被降权。若连续 token 的置信度低于阈值 τ，则截断后续所有损失，避免不可靠信号。
  3. **压缩空间蒸馏（码本松弛）**：将原始码本通过 K-means 聚类压缩为更小的分组，在分组空间上计算 KL 散度。这减少了视觉相似 token 之间的歧义，使蒸馏信号更干净。
  4. **并行解码加速训练**：使用 ZipAR 风格的块并行解码生成学生后缀，提升训练吞吐量约 1.3 倍，不改变推理过程。
- **公式与算法流程**（文字说明）：
  - 标准蒸馏：`L_KD = E_{x<t~p_data}[KL(教师||学生)]`（教师强制）。
  - GKD：`L_GKD = E_{x<t~p_student}[KL(教师||学生)]`（完全学生条件）。
  - VarKD：学生从混合分布 `p_mix`（前 k 个真实 token + 后续学生生成）采样，对后缀位置应用：
    ```
    L_VarKD = E_{x1:T~p_mix, t>k}[ α_{x<t} * KL(˜p_student || ˜p_teacher) ]
    ```
    其中 α 是基于教师熵的置信度，˜p 是压缩空间上的聚合分布。

## 6. 主要结论与发现

1. **标准 KD 已是强基线**，但仅用数据前缀，无法消除训练-推理不匹配。
2. **SeqKD（教师生成序列）效果最差**，因为教师分布与学生分布差异大。
3. **GKD（学生自生成）** 在大容量学生上略有优势，但对小模型不稳定。
4. **VarKD 在所有设置下一致最佳或并列最佳**，尤其在 FID 上提升显著（例如 LlamaGen-B 从 6.51→4.58，L 从 3.07→2.83）。
5. **教师越大未必越好**：中间容量教师（L）效果优于超大教师（XXL），可能因学生容量有限，无法拟合过锐复杂的分布。
6. **两阶段训练（KD 预对齐 + VarKD 微调）** 效率高，VarKD 在 KD 饱和后仍能继续改善。

## 7. 优点

- **首次系统研究**：填补了视觉 AR 模型蒸馏方向的空白，深入分析了语言方法失效的原因。
- **对症下药**：针对视觉 AR 的长序列和 token 歧义设计具体组件（混合上下文、熵重加权、压缩空间），而非简单照搬语言方法。
- **实用性强**：所有训练技巧不影响推理过程，保持标准自回归解码；并行解码加速训练，使方法可行。
- **实验严谨**：统一初始化、多骨干、多教师容量、消融充分，定性定量结合。

## 8. 不足与局限

- **实验场景局限**：仅在 ImageNet 类条件生成上评估，未扩展到开放域文本-图像生成（如 Janus-Pro、DALL·E 等），后者更复杂，要求更强语义对齐。
- **代理可靠性的近似**：教师熵和 token 聚类作为监督可靠性的代理，不一定完全对应语义正确性或 prompt 对齐，在多模态任务中可能不够。
- **计算成本**：VarKD 需要在线学生采样和教师前向传播，训练吞吐量低于 KD（0.20 vs 3.50 steps/s），虽比 GKD 快但仍有开销。
- **依赖已训练的教师**：需要提前训练好大教师模型，这在某些场景下可能不现实。
- **未探讨更复杂的解码策略**：如 speculative decoding 与蒸馏的结合，而仅用训练阶段加速。
