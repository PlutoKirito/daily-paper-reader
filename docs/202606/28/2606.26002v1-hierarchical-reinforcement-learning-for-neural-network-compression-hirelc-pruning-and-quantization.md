## 摘要
我们提出HiReLC，一个用于深度神经网络自动联合量化和结构化剪枝的分层集成强化学习框架。该框架将压缩搜索分解为两个抽象层次：低层智能体（LLAs）在每个块内独立操作，在一个涵盖位宽、剪枝保留比率、量化类型和粒度的多离散动作空间中选择每核配置；而高层智能体（HLAs）则通过基于Fisher信息灵敏度估计的集成投票来协调全局预算分配。为缓解策略评估的计算成本，一个迭代的主动学习循环将代理引导的RL优化与压缩后微调交替进行，使用轻量级MLP代理来分摊昂贵评估，并在冷启动期间使用logit-MSE代理。该代理用于奖励塑形，而非替代最终压缩后评估。控制器设计为架构无关，通过模块化层抽象将RL环境与底层网络拓扑解耦。在Vision Transformer和CNN基准上的实验表明，有效参数存储压缩比达到5.99-6.72倍，在一个设置中准确率提升3.83%，在其他设置中准确率下降0.55-5.62%，支持分层策略分解和灵敏度感知指导作为联合神经网络压缩的实用设计选择。

## Abstract
We present HiReLC, a hierarchical ensemble-reinforcement learning framework for automated joint quantization and structured pruning of deep neural networks. The framework decomposes the compression search across two levels of abstraction: low-level agents (LLAs) operate independently per block, selecting per-kernel configurations over a multi-discrete action space spanning bitwidth, pruning keep-ratio, quantization type, and granularity, while high-level agents (HLAs) coordinate global budget allocation via ensemble voting guided by Fisher Information-based sensitivity estimates. To mitigate the computational cost of policy evaluation, an iterative active learning loop interleaves surrogate-guided RL optimization with post-compression fine-tuning, using a lightweight MLP surrogate to amortize expensive evaluations and a logit-MSE proxy during cold-start. The surrogate is used for reward shaping rather than as a replacement for final post-compression evaluation. The controller is architecture-agnostic by design, with a modular layer abstraction decoupling the RL environment from the underlying network topology. Experiments across Vision Transformer and CNN benchmarks demonstrate effective parameter-storage compression ratios of 5.99 - 6.72$\times$ with a 3.83 % gain in one setting and 0.55 - 5.62 % accuracy drops elsewhere, supporting hierarchical policy decomposition and sensitivity-aware guidance as practical design choices for joint neural network compression.

### 2. 论文提出的方法论：核心思想、关键技术细节

-   **核心思想**：将庞大的联合压缩搜索空间分解为两个抽象层次，分别由不同的RL智能体处理，并辅以灵敏度引导和成本摊销机制。
-   **关键技术细节**：
    -   **分层智能体架构**：
        -   **高层智能体（HLA）**：负责全局预算分配。通过一个集成PPO/A2C代理，为网络中的每个可压缩块（Block）设定目标保留参数比例、最大剪枝率等预算。预算分配受**Fisher信息**估计的块级灵敏度指导。
        -   **低层智能体（LLA）**：负责在高层预算约束下进行每核（Per-kernel）配置搜索。每个块都有一个集成PPO/A2C智能体，在一个四维动作空间（位宽、剪枝保留率、量化类型、量化粒度）中进行搜索。
    -   **Fisher信息灵敏度引导**：利用Fisher信息为每个块计算一个归一的灵敏度分数$s_i$。该分数被用于初始化LLA观测、加入奖励函数中的惩罚项$P_{sens}$，以及指导HLA的预算修正，确保关键块得到更“轻”的压缩。
    -   **迭代主动学习循环**：
        -   **冷启动阶段**：当经验池不足时，使用**Logit-MSE代理**（比较压缩前后模型输出logits的均方误差）作为快速近似，替代代价昂贵的全模型微调。
        -   **代理辅助阶段**：当经验池达到阈值后，训练一个**轻量级MLP代理**来预测给定配置下的模型精度。该代理仅用于RL的**奖励塑形**，最终压缩结果仍需通过完整的微调和评估获得。
        -   **循环迭代**：在每个循环中，先更新HLA状态，再让LLA搜索配置，接着应用压缩配置（先剪枝再量化）、进行微调，最后用新的数据点更新MLP代理。
    -   **架构无关性**：通过一个模块化的层抽象模块，将RL环境与底层网络拓扑解耦，使得该框架可以应用于Vision Transformer和CNN。

### 6. 论文的主要结论与发现

-   HiReLC在几乎所有架构上都实现了高压缩比（5.99-6.72倍），且准确率下降可控（0.55-5.62%），并在CLIP ViT-B/32 on CIFAR-10上甚至实现了准确率提升（+3.83%）。

-   HiReLC是一个架构无关的框架，能适用于ViT和CNN。

### 7. 优点

1.  **方法统一且全面**：首次将分层RL、集成学习、灵敏度分析和主动学习结合成一个统一的自动化联合压缩框架，覆盖了剪枝和量化。
2.  **架构无关性**：控制器设计不依赖特定架构，增加了其通用性，有望成为“一次训练，到处部署”的有力候选。
3.  **搜索效率优化**：通过Logit-MSE代理和MLP代理的冷启动+热切换机制，显著减少了昂贵的完整微调次数，使RL驱动的搜索在计算上更实用。
4.  **灵敏度引导的直观与有效**：将Fisher信息自然融入到奖励函数和动作修正中，提供了一个基于梯度的、重要的块重要性先验，这比简单的基于幅度的启发式方法更有效。
5.  **设计考量细致**：对奖励函数中各分量的数值范围进行了严格推导（见附录A），确保训练信号稳定；并考虑了预算合规、稳定性正则化等细节。

### 8. 不足与局限

1.  **目标单一性**：当前框架仅针对**参数存储大小**（有效参数存储压缩比CR）进行优化，未考虑**延迟（Latency）** 或**能耗（Energy）** 等硬件指标。因此，高CR并不必然带来线性的推理速度提升。
2.  **代理噪声与**：MLP代理在小型CNN（如ResNet18）上MAE高达15.20%，其信号非常嘈杂。作者本身也将代理定位为“粗粒度的排序信号”，在应用时需要谨慎。
3.  **预算合规差距**：实验结果表明，LLA普遍存在过度压缩问题，预算合规率（Budget Compliance）仅在59.2%到62.4%之间，说明奖励函数对预算偏离的惩罚力度尚不足以完全约束智能体行为。
4.  **状态空间简化**：稳定性正则化采用了紧凑的滚动窗口实现，而非完全的马尔可夫状态增广，这可能会引入潜在的偏差。
5.  **实验比较不匹配**：与SOTA方法（如AMC, HAQ）的对比是在**不同数据集和不同协议**下进行的，这削弱了论文的核心数值竞争力。缺乏在ImageNet-1K上的完整、匹配的基准实验是一个主要短板。
