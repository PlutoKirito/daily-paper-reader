## 摘要
扩散 Transformer (DiTs) 在图像和视频生成质量方面表现卓越，但计算成本高昂，限制了其可扩展性和端侧部署。我们提出了 CoReDiT，这是一个适用于各种视觉任务的 DiTs 结构化令牌剪枝框架。CoReDiT 使用线性时间复杂度的空间相干性分数来估计潜在令牌点阵中的局部冗余，并在自注意力机制中跳过高相干性（冗余）的令牌。为了保持稠密表示并避免视觉不连续性，我们通过对空间邻近的保留令牌进行相干性引导的聚合，来重构被跳过的注意力输出。我们还引入了一种渐进式、块自适应的剪枝调度方案，该方案逐渐增加剪枝比例，并为具有更高冗余度的块和去噪步骤分配更大的剪枝预算。在包括 PixArt-α 和 MagicDrive-V2 在内的最先进扩散骨干网络上，CoReDiT 实现了高达 55% 的自注意力 FLOPs 减少，在云端 GPU 上实现了 1.33 倍的推理加速，在移动端 NPU 上实现了 1.72 倍的加速，同时保持了高视觉质量。值得注意的是，CoReDiT 还增加了端侧内存余量，从而支持更高分辨率的生成。

## Abstract
Diffusion Transformers (DiTs) deliver remarkable image and video generation quality but incur high computational cost, limiting scalability and on-device deployment. We introduce CoReDiT, a structured token pruning framework for DiTs across vision tasks. CoReDiT uses a linear-time spatial coherence score to estimate local redundancy in the latent token lattice and skips high coherence (redundant) tokens in self-attention. To maintain a dense representation and avoid visual discontinuities, we reconstruct skipped attention outputs via coherence-guided aggregation of spatially neighboring retained tokens. We further introduce a progressive, block-adaptive pruning schedule that increases pruning gradually and allocates larger budgets to blocks and denoising steps with higher redundancy. Across state-of-the-art diffusion backbones including PixArt-α and MagicDrive-V2, CoReDiT achieves up to 55% self-attention FLOPs reduction and inference speedups of 1.33x on cloud GPUs and 1.72x on mobile NPUs, while maintaining high visual quality. Notably, CoReDiT also increases on-device memory head-room, enabling higher-resolution generation.

### 1. 核心问题与研究动机
*   **核心问题**：扩散 Transformer (DiT) 虽然在图像和视频生成上表现优异，但其自注意力机制的计算复杂度随 Token 数量呈平方级增长（$O(N^2)$），导致在高分辨率生成和移动端部署时面临巨大的计算开销和内存压力。
*   **研究动机**：作者观察到 DiT 生成过程中存在大量空间冗余（如背景、平滑纹理），但现有的 Token 缩减方法存在三个挑战：(1) 识别冗余 Token 的开销过大；(2) 直接丢弃 Token 会导致视觉不连续和伪影；(3) 不同 Transformer 块和去噪步长的冗余度不同，难以确定统一的剪枝策略。

### 2. 方法论
CoReDiT 框架由三个核心组件组成：
*   **空间相干性选择器 (Spatial Coherence Selector)**：
    *   **核心思想**：利用局部相似性衡量冗余。将 Token 划分为非重叠网格，计算每个 Token 与其所属网格平均特征的余弦相似度，定义为“空间相干性分数 (SC)”。
    *   **技术细节**：通过数学变换将计算复杂度从 $O(w^2 N)$ 降低到线性时间 $O(N)$。SC 分数越高，代表该 Token 越能被邻居代表，即越冗余，从而在自注意力计算中被跳过。
*   **相干性引导重构 (Coherence-guided Reconstruction)**：
    *   **核心思想**：为了保持 Token 点阵的完整性，被跳过的 Token 输出通过其邻近保留 Token 的加权聚合来重构。
    *   **技术细节**：使用 SC 分数作为权重进行内容感知插值。为了消除网格边界伪影，采用了交替网格大小和 $m$-步长保留策略（确保每个子网格至少保留一个 Token 作为锚点）。
*   **渐进式块自适应剪枝调度 (Progressive, Block-adaptive Pruning)**：
    *   **核心思想**：在轻量化微调过程中，根据实时统计的冗余度动态分配每个块的剪枝比例。
    *   **技术细节**：采用渐进式增加剪枝量的方法，优先剪枝冗余度最高的块。同时引入两阶段时间步衰减策略，在去噪后期（细节微调阶段）减少剪枝量。

### 6. 主要结论与发现
*   **高效性**：CoReDiT 减少了高达 55% 的自注意力 FLOPs。在 H100 GPU 上实现 1.33 倍加速，在移动端 NPU 上实现 1.72 倍加速。
*   **质量保持**：在大幅减计算量的情况下，FID 指标仅轻微波动，视觉效果几乎无损。
*   **内存优势**：通过 Token 剪枝释放了内存，使得移动端设备能够生成原本会 OOM（显存溢出）的高分辨率图像（如从 1600 提升至 1920 分辨率）。

### 7. 优点
*   **线性复杂度**：SC 分数的计算极其高效，避免了传统注意力权重计算的平方级开销。
*   **硬件友好**：重构算子和剪枝逻辑易于在 NPU 等专用硬件上实现结构化加速。
*   **通用性强**：方法不依赖于特定的 DiT 架构，可广泛应用于图像和视频生成任务。

### 8. 不足与局限
*   **视频时间一致性**：在视频生成实验中，虽然单帧质量保持良好，但 FVD 指标有所下降，说明目前的空间剪枝策略对时间连续性有一定负面影响。
*   **微调依赖**：虽然是轻量化微调，但仍需要一定量的训练数据和算力来学习最优剪枝调度，并非完全的 Plug-and-Play（即插即用）。
*   **非注意力开销**：目前主要针对自注意力机制进行优化，对于 FFN（前馈网络）等其他组件的计算占比随剪枝增加而变得显著，未来需进一步优化。
