## 摘要
生成式视频编解码器（GVC）的最新进展通常将视频编码为二维潜网格（2D latent grid），并采用高容量生成式解码器进行重建。然而，这种范式在充分利用时空冗余方面仍面临两个关键挑战：空间上，由于其僵化的结构，二维潜网格不可避免地保留了帧内冗余，相邻块保持高度相似，从而导致需要更高的码率；时间上，二维潜网格在以紧凑且语义一致的方式建模长期相关性方面效果较差，因为它阻碍了跨帧公共内容的聚合。为了解决这些局限性，我们提出了一种基于一维（1D）潜表示的生成式视频压缩方法（GVC1D）。GVC1D 以短期和长期上下文为条件，将视频数据编码为极度紧凑的一维潜标记（1D latent tokens）。在没有僵化的二维空间对应关系的情况下，这些一维潜标记可以自适应地关注语义区域并自然地促进标记减少，从而降低空间冗余。此外，所提出的一维存储器在保持低计算成本的同时，提供了语义丰富的长期上下文，从而进一步减少了时间冗余。实验结果表明，GVC1D 具有卓越的压缩效率，在 HEVC Class B 数据集上，其在 LPIPS 指标下实现了 60.4% 的码率降低，在 DISTS 指标下实现了 68.8% 的码率降低，超越了以往的视频压缩方法。项目地址：https://gvc1d.github.io/

## Abstract
Recent advancements in generative video codec (GVC) typically encode video into a 2D latent grid and employ high-capacity generative decoders for reconstruction. However, this paradigm still leaves two key challenges in fully exploiting spatial-temporal redundancy: Spatially, the 2D latent grid inevitably preserves intra-frame redundancy due to its rigid structure, where adjacent patches remain highly similar, thereby necessitating a higher bitrate. Temporally, the 2D latent grid is less effective for modeling long-term correlations in a compact and semantically coherent manner, as it hinders the aggregation of common contents across frames. To address these limitations, we introduce Generative Video Compression with One-Dimensional (1D) Latent Representation (GVC1D). GVC1D encodes the video data into extreme compact 1D latent tokens conditioned on both short- and long-term contexts. Without the rigid 2D spatial correspondence, these 1D latent tokens can adaptively attend to semantic regions and naturally facilitate token reduction, thereby reducing spatial redundancy. Furthermore, the proposed 1D memory provides semantically rich long-term context while maintaining low computational cost, thereby further reducing temporal redundancy. Experimental results indicate that GVC1D attains superior compression efficiency, where it achieves bitrate reductions of 60.4\% under LPIPS and 68.8\% under DISTS on the HEVC Class B dataset, surpassing the previous video compression methods.Project: https://gvc1d.github.io/

#### 1. 核心问题与整体含义（研究动机和背景）
传统的生成式视频编解码器（GVC）通常将视频帧编码为**二维潜变量网格（2D Latent Grid）**。这种做法存在两个主要缺陷：
*   **空间冗余：** 2D网格具有僵性的空间结构，每个图像块被强制映射到固定位置的Token，导致相邻Token之间存在大量重复信息，难以充分压缩。
*   **时间建模困难：** 2D结构难以在紧凑的语义空间中聚合跨帧的长期相关性，导致长时上下文（Long-term Context）的存储和检索计算成本高且语义一致性差。
**研究动机：** 探索一种打破2D空间刚性约束的表示方法，通过一维（1D）潜变量实现更紧凑、更具语义代表性的视频压缩。

#### 2. 方法论：核心思想与关键技术
论文提出了 **GVC1D** 框架，其核心流程如下：
*   **1D 潜变量 Token 化：** 
    *   使用基于 Vision Transformer (ViT) 的编码器，将输入帧 $x_t$ 转换为一组无结构、可学习的 **1D 潜变量 Token**。
    *   这些 Token 不再与图像像素建立固定的空间对应关系，而是通过注意力机制自适应地聚合全图的语义和运动信息。
*   **1D 存储机制 (1D Memory)：**
    *   维护一个固定大小的 1D 记忆状态，利用当前帧生成的少量 1D Token 进行循环更新。
    *   由于 1D Token 极其紧凑且语义丰富，该存储器能以极低的计算成本保存长时上下文，并与前一帧的短时特征结合，共同指导当前帧的重建。
*   **自回归熵模型 (AR Entropy Model)：**
    *   采用基于 Transformer 的自回归模型对量化后的 1D Token 进行概率估计。由于 Token 数量极少（如每窗口仅 32 个），自回归推理的计算开销在可控范围内。
*   **生成式解码：**
    *   解码器利用 1D Token 和时空上下文，通过强大的生成能力恢复出具有丰富细节的视频帧。

#### 7. 优点（亮点）
*   **范式创新：** 首次在视频压缩中引入 1D 潜变量表示，打破了长久以来对 2D 空间结构的依赖。
*   **极度紧凑：** 能够用极少量的 Token（如每帧仅几十个）代表复杂的视频内容，极大地压缩了瓶颈层。
*   **感知质量高：** 在极低码率下依然能保持优秀的视觉细节，减少了传统方法常见的模糊现象。

#### 8. 不足与局限
*   **保真度权衡：** 作为一个生成式模型，其 PSNR 指标仍低于专门优化 PSNR 的非生成式模型（如 DCVC-FM），这意味着它在像素级精确度上可能存在偏差。
*   **应用范围限制：** 目前主要针对**极低码率的有损压缩**。由于 Token 数量极少，该方法可能难以扩展到需要高保真或无损压缩的专业场景。
*   **复杂度：** 虽然 Token 数量少，但基于 Transformer 的全局注意力机制在处理超高分辨率视频时，计算复杂度仍是一个潜在挑战。
